Icebreaker is a high-performing **data lakehouse** built on the PAID stack:

- Apache <a href="https://parquet.apache.org/" target="_blank">Parquet</a>: a high performance, columnar data file format for big data 
- Apache <a href="https://arrow.apache.org/" target="_blank">Arrow</a>: a columnar format for fast data interchange and in-memory analytics
- Apache <a href="https://iceberg.apache.org/" target="_blank">Iceberg</a>: an open table format (OTF) for big data analytics
- Apache <a href="https://datafusion.apache.org/" target="_blank">DataFusion</a>: an extensible query engine written in Rust that uses Apache Arrow as its in-memory format
